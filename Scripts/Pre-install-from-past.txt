1:	R-Packages to install
	* R2jags
	* xtable
	* R2WinBUGS
	* Hmisc

To install these packages simply run the below line from the command line in an R session.  
install.packages(c("R2jags","xtable","R2WinBUGS","Hmisc"))

Depending on what you have already installed this may install numerous other packages that these packages depend upon.
